# PhysioGuardian  
### *Your Pocket Physiotherapist - Simple, Smart & Helpful.*

PhysioGuardian is a lightweight and user-friendly physiotherapy assistant that helps users understand their pain and instantly get a personalized exercise plan. No login, no confusion just describe your pain and start recovering.
